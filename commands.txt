pwd -- print working directory
	> display your current directory location

cd [loc] -- change directory
	> allows you to move throughout your computer

mkdir [folder name] -- used to make new folder

ls -- display the contents of your current directory

touch [file name] -- used to create a file in the current directory

rm [file name] -- used in deleting and removing of a file

repository -- folder or container that stores a files or folders related to a project, also keeps track of the changes made overtime.
>local repository -- version of the repository stored in your local computer
>remote repository -- version of the repository stored in the server/GitHub

git init -- used to initialize a folder

git config --global user.name "<git hub username>"

git config --global user.email "<GitHub email">

to check if the user name and email address and username:
	git config --global user.name
	git config --global user.email

git add . -- adds all modified files in your current directory to the staging area

git add <file name> -- adds a specific file only to the staging area

git  commit -m "message" -- saves the changes that you've made to the files in the staging area to the repository. 			The message describes what changes were made in this commit. 

git status -- displays the current status of your file/folder

git log -- show all the commits that were made

git log --oneline  -- to show the short version of all the commits that we make
